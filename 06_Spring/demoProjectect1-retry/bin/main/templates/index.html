        from this class there is a main method
        -> run -> Tomcat server

Project Structure

  • src/main/java : Java source code
  • src/main/resources : resources of the project (preferences/css/html etc...)
    • template : Java templates (.html extention : thymeleaf)
    • static : static resources for templates (JS/CSS/img etc...)
    • application.properties : Spring Boot project -> global preferences
  • Project & External Dependencies : build.gradle -> lib
  • build.gradle : gradle project -> lib/build/deployment
